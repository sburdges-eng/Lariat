# Daily Cleaning

Daily cleaning is folded into station closing. This page exists so anyone can
look it up in isolation — auditors, new hires, a closer who's never done this
before.

---

## Line stations (every close)

- [ ] Restock / top-fill station
- [ ] Flip station
- [ ] Refill / replace sauce + squeeze bottles
- [ ] Wrap line (garde + fry partner on light nights)
- [ ] Wipe down cutting boards
- [ ] Sanitize prep sink, knives, surfaces
- [ ] Clean stovetop, char-broil, flat top (grill/saute)
- [ ] Season flat top (grill/saute)
- [ ] Clean / filter fryer oil (fry)
- [ ] Ice-bath + wrap steam well contents (expo/saute)
- [ ] Empty grease trays (grill/saute)

## House (end of night)

- [ ] Shut off steam well, grill, fryer, oven
- [ ] Restock reach-in
- [ ] Wipe down stainless surfaces, cutting boards
- [ ] Wash + put away dishes; drain dishwasher
- [ ] Sweep + mop floor; empty mop bucket
- [ ] Shut off gas line to wok station
- [ ] Shut off hood vent

## Sign-off

The iPad: `/admin/cleaning-schedule` → Daily → sign off.
House-close sign-off also doubles as a daily cleaning attestation.
