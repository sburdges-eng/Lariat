# Closing — Salad / Garde

> Source: `CLOSING SALAD.xlsx` + `Closing checklist_ Positions_.docx`

---

## 7:00 PM — first pass

- [ ] Restock station
- [ ] Flip station
- [ ] Clean out prep sink (including remaining knives from opening)
- [ ] Take out station trash

## 9:00 PM — shutdown

- [ ] Wrap line (protein and salad)
- [ ] Put cutting board in dish pit
- [ ] Wipe down stainless surfaces
- [ ] Sweep entire line
- [ ] Check GFI outlets
- [ ] Turn off sink
- [ ] **LINE CHECK** — final closing line check
- [ ] Clock out

---

## Things that must happen every close

- [ ] Restock / top-fill station
- [ ] Flip station
- [ ] Refill / replace sauce bottles
- [ ] Wrap line (garde + fry share this on light nights)
- [ ] Wipe down cutting boards (bleach every few days)
- [ ] Sanitize prep sink / knives / surfaces
