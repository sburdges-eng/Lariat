**Button** — compact, uppercase, wide-tracked action control. Matte fill, 1px hairline that lights to amber on hover, scales down on press.

```jsx
<Button variant="primary">Save</Button>
<Button>Count</Button>
<Button variant="ghost">Go back</Button>
<Button variant="danger">86 it</Button>
<Button variant="ok" size="lg">Done</Button>
```

- Variants: `primary` (amber), `default` (matte), `ghost`, `ink` (bone), `danger` (oxblood), `ok` (sage).
- Sizes: `xs`, `sm`, `md` (default), `lg`.
- Copy uses kitchen verbs — Start, Done, Need, Out, Count — never "Submit"/"Configure".
